# EcoTrack Frontend

EcoTrack Frontend is a web-based application for environmental tracking and monitoring.

## Project Structure

```
frontend/
├── index.html              # Main application page
├── pages/
│   └── map-tracker.html    # Map tracking page
├── css/
│   ├── desktop.css         # Desktop styling
│   └── main.css            # Main stylesheet
├── app-js/
│   ├── device-tracker.js   # Device tracking logic
│   └── map.js              # Map functionality
└── assets/
    └── icons/              # Application icons
```

## Features

- Device tracking and monitoring
- Interactive map visualization
- Real-time location updates
- Responsive design for desktop

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Files in ZIP

The `ecotrack-frontend.zip` contains:

1. **README.md** - Documentation
2. **index.html** - Main HTML entry point
3. **css/desktop.css** - Primary styling
4. **css/main.css** - Additional styles
5. **app-js/device-tracker.js** - Device tracking logic (Vanilla JS)
6. **app-js/map.js** - Map functionality (Vanilla JS)
7. **pages/map-tracker.html** - Dedicated map tracking page
8. **assets/icons/** - Icon assets folder

## How to Use

1. Extract the ZIP file
2. Open `index.html` in a web browser
3. Navigate through the application
4. Use map-tracker page for location tracking

## Installation

No build process required. This is a static frontend application.
