# EcoTrack AI: Autonomous Device Finder 📱

EcoTrack AI is a smart and autonomous mobile application designed to locate misplaced or lost smartphones. It utilizes 'on-device intelligence' to function effectively even without an active internet connection.

## 🚀 Core Features
* **Voice Wake-Word:** Triggers a loud alarm via a preset voice command ("Where is my phone?"), bypassing silent mode even without internet.
* **Last-Breath SOS:** Automatically captures precise GPS location and sends an emergency SMS/Email to trusted contacts the exact moment the battery hits 1%.
* **Intelligent Proximity:** Uses Bluetooth and Wi-Fi signal strength (RSSI) to guide the user to the device's approximate location.

## 🛠 Tech Stack
* **Language:** Kotlin
* **Framework:** Android SDK
* **AI/ML:** TensorFlow Lite (Offline Keyword Spotting)
* **Tools:** Android Studio, Cursor/GitHub Copilot

## 💡 Why EcoTrack AI?
Unlike general assistants like Siri or Google Assistant, EcoTrack AI:
1. Functions entirely **Offline**.
2. Operates during critical battery-critical scenarios (**Last-Breath SOS**).
3. Acts as a 'Specialized Security Agent' rather than a general assistant.
