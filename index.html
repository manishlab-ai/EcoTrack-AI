Core Features
Offline Voice Wake Detection
TensorFlow Lite model listens for wake phrase even without internet. Triggers max-volume alarm + strobe light to locate the device in the dark or silent mode.
Intelligent Proximity Finder (BLE)
Uses Bluetooth Low Energy RSSI values to calculate distance. Displays a real-time Hot/Cold radar UI — no internet required at all.
Last Breath SOS
When battery hits 1%, a BroadcastReceiver fires automatically — blasting high-accuracy GPS coordinates via SMS to all emergency contacts. Works offline.
Theft Detection (Accelerometer)
Monitors sudden accelerometer changes to track theft risk score. Abnormal movement patterns trigger silent alert and GPS snapshot.
Safe Zone Intelligence
Google Play Geofencing API detects when you leave your device behind in a defined safe zone. Immediate push notification + email alert dispatched.

What is EcoTrack AI?

EcoTrack AI is an autonomous device guardian by Sentinel-Logic. It protects, tracks, and manages your devices in real time — even offline, at 1% battery, or with SIM removed. Built for the FAR AWAY Hackathon.

Getting Started
1. Sign Up / New Registration
Fill Organization, Role, Device Name, password. Verify email.
2. Add Emergency Contacts
Settings → Emergency Contacts → Add email → Verify with OTP.
3. Set Smart Routines
Routine tab → Add Routine → Set time, days, mode type.
4. Monitor Devices
Device tab → view status, send Ring/Lock/Erase commands.
